# La Mejor Abogada — Aventura

Un juego de plataformas estilo Mario Bros hecho con [Phaser 3](https://phaser.io).

## Cómo jugar

- **← →** mover
- **ESPACIO / ↑** saltar (con doble salto en el aire)
- **F** disparar bolas de fuego (cuando tienes la flor)
- **↓** agacharse / entrar a tubos warp
- **M** silenciar

En móvil aparecen botones táctiles automáticamente. Hay que jugar en horizontal.

## Power-ups

- 🍄 Champiñón — te hace grande
- ⭐ Estrella — invencibilidad temporal
- 🌻 Flor — disparas bolas de fuego

## Local

Cualquier servidor estático funciona, por ejemplo:

```bash
python -m http.server 8765
```

Y abre <http://localhost:8765>.
